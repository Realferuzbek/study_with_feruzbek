# StudyMate (Focus Squad) Overview

StudyMate (Focus Squad) is a study focus app that helps you plan, track, and stay accountable. It combines a focus timer, task planning, motivation prompts, live study sessions, and competitive leaderboards.

Key areas you can use:
- Dashboard for quick access to everything.
- Timer to run focus and break blocks.
- Task Scheduler to plan tasks and due dates.
- Motivation Vault for the daily mantra.
- Live Stream Studio for live and scheduled sessions.
- Leaderboard for daily, weekly, and monthly rankings.
- Ask AI for site-specific guidance.

Guests can explore public features and guidance. Signed-in users can access personal tasks, sessions, and progress.
